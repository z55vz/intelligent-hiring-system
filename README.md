# 🎯 Intelligent Hiring System

AI-powered resume ranking system using semantic embeddings (BGE-Small).

## 🚀 Live Demo
👉 [Launch App](https://your-app.streamlit.app)

## ✨ Features
- 📦 Bulk PDF parsing from ZIP uploads
- 🧠 Semantic ranking via Sentence-Transformers
- 🎛️ Mandatory + soft skill filtering
- 📊 Interactive analytics dashboard
- 💾 Cached embeddings for instant re-ranking

## 🛠️ Local Development
\`\`\`bash
pip install -r requirements.txt
streamlit run app.py
\`\`\`

## 🏗️ Tech Stack
Streamlit • PyMuPDF • Sentence-Transformers • scikit-learn • Plotly
